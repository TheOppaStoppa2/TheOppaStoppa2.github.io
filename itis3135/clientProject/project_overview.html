Project Overview

This web application promotes the New York Mets to the general public by providing fans with news, events, and interactive content to increase engagement and attract new fans.

Intended users include sports enthusiasts, current and potential Mets fans, and the general public interested in baseball.

The website will include information about the team, game schedules, player profiles, an event calendar, galleries of photos and videos, and interactive forms for fan engagement.

Page Designs

Home Page

  • Purpose: Welcome users and provide overview of the Mets and fan engagement opportunities
  • Audience: All users
  • Content: Intro text, latest news, featured player, links to other pages
  • Forms: None
  • Buttons/Links: Navigation bar, links to articles and events
  • Actions: Clicking links navigates to other pages

About Page

  • Audience: All users
  • Content: Team history, mission statement, player bios
  • Forms: None
  • Buttons/Links: Navigation links, social media links

Gallery Page

  • Purpose: Showcase photos and videos of games and events
  • Audience: Fans
  • Content: Image gallery, video clips
  • Forms: None
  • Buttons/Links: Clickable images open lightbox or modal
  • Actions: JavaScript slideshow and image modal

Events Page

  • Purpose: Show upcoming games and fan events
  • Audience: Fans, general public
  • Content: Event calendar, ticket info
  • Forms: Optional RSVP forms
  • Buttons/Links: Event links, navigation
  • Actions: Form validation using JavaScript

Contact Page

  • Purpose: Allow fans to reach out with questions or feedback
  • Audience: Fans and general public
  • Content: Contact form with fields for name, email, message
  • Forms: Name, email, message
  • Validation: Email format, required fields
  • Buttons/Links: Submit button
  • Actions: JavaScript validation on form submit

Dynamic Functionality

  • Gallery Page: JavaScript slideshow to display game photos. Example: W3Schools Slideshow
  • Contact Page: JavaScript form validation to check required fields and valid email format.
  • Events Page: Interactive calendar using jQuery UI Datepicker to show upcoming events. Example: jQuery UI Datepicker
